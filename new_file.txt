This is just an example file. This also applies when changing files!

We'll use the following commands to add this file (and another other changes we've made to our GIT

	git add .

	git commit -m "made some changes"
	
	git push -u origin [branch name]

So, in this branch, for the first time you clone it, you'll type:

	git push -u origin Tara-2

But, in all subsequent pushes, all you need to type is:

	git push

Then, enter your credentials. You may need to do this a few times, as you did previously. Then you're all good.
